A number is called a Happy Number if:
Take the number
Replace it with the sum of squares of its digits
Repeat the process
If it becomes 1 →  Happy Number
If it enters a loop (never reaches 1) →  Not Happy

 In coding, we stop when:

num == 1 → Happy
num == 4 → Not Happy (because 4 is part of a loop)
 Example 1 (Happy Number)

Input: 19

1² + 9² = 1 + 81 = 82  
8² + 2² = 64 + 4 = 68  
6² + 8² = 36 + 64 = 100  
1² + 0² + 0² = 1 ✔

 Output: Happy Number

 Example 2 (Not Happy Number)

2² = 4  
4² = 16  
1² + 6² = 37  
3² + 7² = 58  
5² + 8² = 89  
8² + 9² = 145  
1² + 4² + 5² = 42  
4² + 2² = 20  
2² + 0² = 4 → Loop 

Output: Not Happy Number

 Example 3 (Happy Number)

Input: 7

7² = 49  
4² + 9² = 16 + 81 = 97  
9² + 7² = 81 + 49 = 130  
1² + 3² + 0² = 1 + 9 + 0 = 10  
1² + 0² = 1 ✔

 Output: Happy Number

 Example 4 (Not Happy Number)

2² + 0² = 4 → Loop starts

Definition

A Strong Number is:
A number that is equal to the sum of the factorials of its digits

 Simple Explanation
Take each digit
Find its factorial (!)
Add all factorials
If sum = original number →  Strong Number
 Example 1 (Strong Number)
Input: 145
1! + 4! + 5! = 1 + 24 + 120 = 145 ✔
 Output: Strong Number

 Example 2 (Strong Number)
Input: 2
2! = 2 ✔
Output: Strong Number

 Example (Not Strong)

1! + 2! + 3! = 1 + 2 + 6 = 9 ≠ 123
Output: Not Strong Number

Create a base class Animal with a method getLifespan() that returns 0.
 Create a derived class Elephant that overrides getLifespan() to return a given lifespan.
 Use dynamic method dispatch to call getLifespan() on an Animal reference pointing to an Elephant.
Input Format
An integer – lifespan in years.

 Output Format
Print the lifespan.

Testcase 1
Input
70

Output
70

Explanation:
 Elephant’s overridden getLifespan() returns 70.

Testcase 2
Input
45

Output
45

Explanation:
 Animal animal = new Elephant(45); → method dispatch calls subclass version.

Testcase 3
Input
100

Output
100

Explanation:
 Overridden method returns the provided lifespan value.
