Take the number
Replace it with the sum of squares of its digits
Repeat the process
If it becomes 1 →  Happy Number
If it enters a loop (never reaches 1) →  Not Happy

 In coding, we stop when:

num == 1 → Happy
num == 4 → Not Happy (because 4 is part of a loop)
 Example 1 (Happy Number)

1² + 9² = 1 + 81 = 82  
8² + 2² = 64 + 4 = 68  
6² + 8² = 36 + 64 = 100  
1² + 0² + 0² = 1 ✔

3² + 7² = 58

4² + 2² = 20  
2² + 0² = 4 → Loop 

Output: Not Happy Number

4² + 9² = 16 + 81 = 97  
9² + 7² = 81 + 49 = 130  
1² + 3² + 0² = 1 + 9 + 0 = 10  
1² + 0² = 1 ✔

 Output: Happy Number

A Strong Number is:
A number that is equal to the sum of the factorials of its digits

 Simple Explanation
Take each digit
Find its factorial (!)
Add all factorials
If sum = original number →  Strong Number
 Example 1 (Strong Number)

 Output: Strong Number

 Example 2 (Strong Number)
Input: 2

Output: Strong Number

 Example (Not Strong)

Output: Not Strong Number
